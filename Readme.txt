Отправка формы на JS/JQ:
Для начала нужно подключить callback.js и style_load.css
В load.html лежит готовый вариант модального окна загрузки

1. Создать форму, в <input> из которого хотите брать занчение, добавьте класс form_field, а также аттрибут data-name.
form_field - скрипт находит все инпуты которые есть в форме, но значения будет брать из именно этих(не добавлять чекбоксам, радио и скрытым инпутам, у них свои индентификаторы)
data-name - чтобы понимать какое значение за что отвечает, мы задаем в data-name название (Например: data-name="Имя: " в <input> пользователь введет "Игорь", получится "Имя: Игорь")
2. Если в форме есть чекбоксы и их нужно отобразить, добавляем класс radio-checkbox
3. Если в форме есть скрытые инпуты и их нужно отобразить, добавляем класс inputHide и задаете ему имя через аттрибут name.
4. В файле callback.js вместо chatid и token вставьте свой ИД чата и токен бота

Выводится форма будет в определенном порядке:
1.Скрытый инпут
2.Чекбоксы
3.Сами инпуты

Для создания своего блока загрузки делаете следующее:
1.Создаете блок с классом modal_load в любом месте(!!!Блок должен быть скрытым, используйте display:none).Это будет наше модальное окно
2.Создаете внутри modal_load еще один блок с классом load(Блок вашего загрузчика)
3.Внутри load создаете блоки для загрузки и с помощью css анимируете, в style_load.css есть готовый вариант можете изменить если хотите
